# 🎹 Piano

An aesthetic piano you can play right in your browser! ✨🌸

## 🌐 Live Demo
[Click here to play!](https://piano-nuxo.onrender.com)

## 🧚 Features
- Play notes by clicking the keys or using your keyboard
- Fairy theme with sparkles and animations
- Works on mobile too (tap the keys!)
- White keys: A S D F G H J
- Black keys: W E T Y U

## 🛠️ Built With
- Python & Flask
- HTML & CSS
- JavaScript (Web Audio API for sounds)

## 🚀 Run Locally
```bash
pip install flask
python app.py
```
Then open http://127.0.0.1:5000
